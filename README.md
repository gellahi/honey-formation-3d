An immersive 3D animated website that simulates the complete journey of honey formation, from bee foraging to bottled honey.

## Features

- **5 Interactive Stages**:  Bee Foraging ‚Üí Honeycomb Building ‚Üí Nectar Processing ‚Üí Harvesting ‚Üí Bottling
- **3D Graphics**:  Powered by Three.js with smooth animations
- **Educational Content**: Learn about each stage of honey production
- **Scroll Navigation**: Scroll through the honey-making journey
- **Responsive Design**: Works on desktop and mobile devices

## How to Use

1. Open `index.html` in a modern web browser
2. Scroll down to navigate through the stages
3. Click "Next Stage" buttons to jump between sections
4. Hover over elements for additional information

## Technologies

- Three.js for 3D rendering
- Vanilla JavaScript for interactivity
- CSS3 for styling and transitions
- HTML5 for structure
